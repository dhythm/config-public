# Commit and Push

## Instructions

1. `git add -A`
2. `git --no-pager diff`
3. Think commit_message in English from diff
4. `git commit -m <commit_message>`
5. `git push -u origin`

## Guideline

- コミットメッセージは修正内容を元に考えてください。
- follow Conventional Commits
